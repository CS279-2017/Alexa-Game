# Alexa Party Games Application

[![PartGames.jpg](https://s24.postimg.org/hih94fk6t/Part_Games.jpg)](https://postimg.org/image/4eborqs4x/)

Party Games is an Alexa application for the Amazon Echo. The application will allow you to play a multitude of voice interaction games against Alexa, or your friends. The following games will be supported upon release:

  - Hangman
  - Fortunately, Unfortunately
  - Word Train
  - Trivia Challenge
